# Monarch R

**monarchr** is an R package for accessing the Monarch Knowledge Graph from R.
It provides functions for searching, retrieving entities, and exploring
associations within the Monarch KG.

## Documentation

Full documentation for the monarchr package is available at:

**[https://monarch-initiative.github.io/monarchr/](https://monarch-initiative.github.io/monarchr/index.html)**

## Source Code

The monarchr source code is hosted on GitHub:

**[https://github.com/monarch-initiative/monarchr](https://github.com/monarch-initiative/monarchr)**

## Installation

```r
# Install from GitHub
devtools::install_github("monarch-initiative/monarchr")
```

See the [monarchr documentation](https://monarch-initiative.github.io/monarchr/index.html)
for detailed installation instructions and usage examples.
