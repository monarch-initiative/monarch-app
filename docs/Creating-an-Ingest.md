# Creating an Ingest

Monarch ingests are built as standalone repositories using Koza.

To create a new ingest, use the
[koza-ingest-template](https://github.com/monarch-initiative/koza-ingest-template)
cookiecutter, which provides the standard project structure, CI configuration,
and testing setup.

See the [Koza documentation](https://koza.monarchinitiative.org/) for details
on writing transforms, configuring downloads, and testing.

For questions about proposing a new data source for inclusion in the Monarch KG,
open an issue on the relevant ingest repository or reach out on the
[Monarch Slack](https://monarchinitiative.slack.com/).
